# Walmart Sales Data Analysis

## Project Overview
This project analyzes Walmart sales data using SQL to understand store performance, holiday impact, and sales trends over time.

## Dataset
The dataset contains weekly sales data for multiple Walmart stores, including:
- Store
- Date
- Weekly_Sales
- Holiday_Flag
- Temperature
- Fuel_Price
- CPI
- Unemployment

## Tools Used
- SQLite
- Excel
- GitHub

## Analysis Performed
- Store-wise total sales analysis
- Holiday vs non-holiday sales comparison
- Monthly sales trend analysis
- Average weekly sales per store
- Holiday impact on average weekly sales

## Key Insights
- Certain stores consistently outperform others in total sales.
- Monthly aggregation reveals seasonal sales patterns.
- Average sales are higher during holiday weeks compared to non-holiday weeks.

## How to Run
1. Import the dataset into SQLite
2. Run the queries from `walmart_sales_analysis.sql`
